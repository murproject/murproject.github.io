# MUR Project

Добро пожаловать в репозиторий проекта **MUR** (Micro Underwater Robot)

## О проекте

**MUR Project** - это открытая платформа для разработки подводной робототехники, созданная Центром Робототехники. Проект включает в себя аппаратные и программные решения для создания подводных аппаратов различного назначения.
